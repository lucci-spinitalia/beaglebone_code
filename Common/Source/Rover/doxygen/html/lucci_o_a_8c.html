RTBvector OA_perform_avoidance ( RTB_FLOAT_TYPE  readings[],
long  count,
RTBvector  yaw,
RTBvector  desired_heading 
)

Perform avoidance.

This function must be periodically called to guarantee correct recording of positions. It is designed to be integrated in a superloop. If the module is disabled, calling this function won't affect performance, as no operations will be performed in such a case (the module will check its state and immediately return).

Parameters:
readingsvector of RTB_FLOAT_TYPE values carrying the laser readings
countlong variable containing the number of laser readings in a vector
yawRTBvector containing the current yaw (true north) of the vehicle
desired_headingRTBvector containing the desired heading
Returns:
an RTBvector with the actual commands (heading, speed)
See also:
RTBvector
