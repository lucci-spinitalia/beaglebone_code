#define RTB_FLOAT_TYPE   double

RTB float type mask definition.

To make the software as much platform agnostic as possible, this definition allows for custom remapping of the float type to whatever deemed fit. Even integer types could be used, but this would surely cause problems with the mathematical operations involved.

RTBvector  vect

Vector set norm.

Arbitrarily set the norm of a vector. Useful to modify the speed should it be connected with the length of the vector. Vector value must be fully qualified: both X and Y together with current NORM must be coherent values. Angles get recalculated on the fly.

Parameters:
normRTB_FLOAT_TYPE carrying the new norm
