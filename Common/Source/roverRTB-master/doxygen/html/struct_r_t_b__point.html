Detailed Description

Waypoint struct (Cartesian)

Used in a list, each element holds essential information for the waypoints acquired during operation.

Parameters:
xMember 'x' contains the X value of the point.
yMember 'y' contains the Y value of the point.
distance_from_startMember 'distance_from_start' contains the distance of the point from the origin.
nextMember 'next' contains a pointer to the next waypoint (farther from the origin).
previousMember 'previous' contains a pointer to the previous waypoint (nearer to the origin).
